### IFileViewer 인터페이스

```csharp
public interface IFileViewer
{
    UIElement          View         { get; }
    ViewerCapabilities Capabilities { get; }

    void Load(string filePath);
    void ApplyTheme(AppTheme theme);
    void Export(ExportFormat format, string destPath);

    event EventHandler<string> TitleChanged;
    event EventHandler<bool>   ModifiedChanged;
}

[Flags]
public enum ViewerCapabilities
{
    ReadOnly = 0,
    Edit     = 1 << 0,
    Export   = 1 << 1,
}

public enum ExportFormat { Html, Pdf }
```

## C# ↔ JS 통신

### C# → JS (ExecuteScriptAsync)

모든 인자는 `JsonSerializer.Serialize()`로 이스케이프 후 전달한다.

| 호출 | 시점 |
|------|------|
| `app.open(content, theme, mode)` | 파일 로드 |
| `app.setTheme(theme)` | 테마 변경 |
| `app.markSaved()` | 저장 완료 후 |

### JS → C# (postMessage / WebMessageReceived)

| 메시지 type | 데이터 | C# 처리 |
|-------------|--------|---------|
| `modified` | `{value: bool}` | `ModifiedChanged` 이벤트 → 탭 타이틀 `*` |
| `save-request` | `{content: string}` | `Save(content)` 호출 |
| `export-html` | `{content: string}` | SaveFileDialog → 파일 저장 |
| `open-link` | `{uri: string}` | 링크 화이트리스트 처리 |

TOC 추출, 활성 항목 추적, 드래그 리사이즈는 JS 내부에서 완결되므로 postMessage 없음.

## 저장 처리

### Ctrl+S 흐름

```
[JS] Ctrl+S 감지
    → postMessage({type:'save-request', content: editor.getValue()})

[C# WebMessageReceived]
    → Save(content)
        → _watcher.EnableRaisingEvents = false   // 저장 중 감시 중단
        → File.WriteAllText(tmp, content, _encoding)
        → File.Replace(tmp, _filePath, backup:null)  // 원자적 교체
        → _watcher.EnableRaisingEvents = true
        → ExecuteScriptAsync("app.markSaved()")

[JS] app.markSaved()
    → 수정 상태 초기화
    → postMessage({type:'modified', value:false})
```

### 파일 인코딩

Load() 시 BOM 바이트로 인코딩을 감지하고, 저장 시 동일한 인코딩을 사용한다.

```csharp
Encoding DetectEncoding(string filePath)
{
    var bom = new byte[4];
    using var fs = File.OpenRead(filePath);
    fs.Read(bom, 0, 4);

    if (bom[0] == 0xEF && bom[1] == 0xBB && bom[2] == 0xBF) return Encoding.UTF8;
    if (bom[0] == 0xFF && bom[1] == 0xFE) return Encoding.Unicode;
    if (bom[0] == 0xFE && bom[1] == 0xFF) return Encoding.BigEndianUnicode;
    return new UTF8Encoding(false);
}
```

### 수정 상태 표시

JS가 `postMessage({type:'modified', value:true})`를 보내면 ViewerHost가 AvalonDock 탭 타이틀에 `*`를 붙인다.

```
README.md  →  README.md *
```

### 닫을 때 미저장 확인

수정 상태가 true인 뷰어를 닫으려 하면 기존 `PromptWindow`로 확인한다.

```
"README.md에 저장하지 않은 변경 사항이 있습니다."
[저장 후 닫기]  [저장 안 함]  [취소]
```

### 외부 변경 감지

FileSystemWatcher `Changed` 이벤트에 200ms 디바운스를 걸어 이벤트 폭풍을 방지한다.

- **미수정 상태**: 자동 리로드
- **수정 중**: 배너 표시 → [디스크 버전으로 되돌리기] / [무시]

## 마크다운 지원 범위

### 지원됨

| 요소 | 문법 예시 | 처리 방식 |
|------|-----------|-----------|
| 헤딩 H1–H6 | `# 제목` | marked.js → HTML + CSS |
| 단락·줄바꿈 | 빈 줄 / 공백 2개 | |
| 굵게·기울임·취소선 | `**b**` `*i*` `~~s~~` | marked.js GFM |
| 인라인 코드 | `` `code` `` | CSS 배경 박스 |
| 코드 블록 구문 강조 | ` ```csharp ``` ` | marked.js + highlight.js (170+ 언어) |
| 인용문 | `> text` | CSS 좌측 강조 바 |
| 목록 (순서/무순서/중첩) | `- item` `1. item` | |
| 체크박스 목록 | `- [ ] / - [x]` | `<input type="checkbox" disabled>` |
| 수평선 | `---` | `<hr>` |
| 링크 | `[text](url)` | `open-link` postMessage → C# 처리 |
| 이미지 | `![alt](path)` | 로컬 파일 → `file:///` URI |
| 표 | GFM `\| col \|` | `<table>` + CSS |
| YAML front matter | `---\ntitle:...\n---` | 문서 상단 메타 박스 |
| Mermaid 다이어그램 | ` ```mermaid ``` ` | mermaid.js 로컬 번들 |
| LaTeX 수식 | `$...$` `$$...$$` | KaTeX 로컬 번들 |
| 각주 | `[^1]` | marked-footnote 플러그인 |
| 이모지 단축코드 | `:smile:` | marked-emoji 플러그인 |

### 제약 있는 요소

| 요소 | 제약 |
|------|------|
| Raw HTML 블록 | `marked.use({allowDangerousHtml: false})` 기본 차단. 설정에서 허용 옵션 추후 제공 가능. |
| 이미지 (외부 URL) | `WebResourceRequested`로 외부 도메인 차단. 로컬 파일만 표시. |
| 내부 `.md` 링크 | `open-link` postMessage → C#에서 ViewerHost.OpenFile() 호출. |
| Mermaid 복잡 다이어그램 | mermaid.js 번들 약 3MB. 문서당 첫 로드 시 파싱 비용 있음. |

### Phase 02-E — Export

function exportHtml() {
    const style  = document.getElementById('theme-link').sheet.cssText ?? '';

    const full   = `<!DOCTYPE html><html><head><meta charset="utf-8">
                    <style>${style}</style></head><body>${body}</body></html>`;
    window.chrome.webview.postMessage({type: 'export-html', content: full});
}
```

| # | 항목 | 적용 위치 |
|---|------|-----------|
| 1 | `ExecuteScriptAsync`에 넘기는 모든 인자를 `JsonSerializer.Serialize()`로 이스케이프 | `MarkdownViewer.xaml.cs` 전체 |
| 2 | 링크 프로토콜 화이트리스트 — http/https만 `Process.Start`, 나머지 차단 | `HandleLinkClick()` |
| 3 | `marked.use({allowDangerousHtml: false})` 명시 | `markdown-app.html` |
| 4 | `mermaid.initialize({securityLevel: 'strict'})` 명시 | `markdown-app.html` |
| 5 | `WebResourceRequested`로 가상 호스트 외 외부 요청 차단 | `InitializeWebView()` |
| 6 | BOM 감지로 인코딩 결정, 저장 시 동일 인코딩 유지 | `DetectEncoding()` / `Save()` |
| 7 | 저장 중 `FileSystemWatcher.EnableRaisingEvents = false` | `Save()` |
| 8 | `await EnsureCoreWebView2Async()` 완료 후 `Load()` 실행 | `InitializeWebView()` |
