# 콘솔 설정

## 개요

Folderss의 콘솔 설정은 기본으로 사용할 커맨드라인과 추가 실행 항목을 관리하는 용도다.

## 설정 항목

### 1. 콘솔 디폴트 커맨드라인

다음 기본 항목 중 하나를 기본값으로 선택할 수 있다.

- PowerShell 7
- Windows PowerShell
- 명령 프롬프트

저장 후 새 콘솔 탭을 열거나 기존 콘솔을 다시 시작하면 선택한 항목 기준으로 실행된다.

### 2. 추가 항목 관리

기본 제공 항목 외에 사용자 정의 실행 항목을 등록할 수 있다.

각 항목은 다음 정보를 가진다.

- 표시 이름
- 셸 종류
- 실행 파일 경로
- 인수

등록한 항목은 `콘솔 디폴트 커맨드라인` 선택 목록에도 함께 표시된다.

## 제외 항목

현재 문서에서는 콘솔 폰트 크기 설정을 안내 대상에서 제외한다.
